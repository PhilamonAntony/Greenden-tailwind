# Greenden-tailwind

Greenden is a modern, responsive plant store website built using HTML5 and Tailwind CSS. The project focuses on creating a clean, user-friendly interface with a mobile-first responsive design. It showcases a collection of plants, featured products, customer testimonials, and a contact section, making it a great front-end practice project.

✨ Features
Responsive design for desktop, tablet, and mobile devices
Modern and clean UI using Tailwind CSS
Hero section with call-to-action
Featured plant collections
Customer reviews/testimonials
Contact section
Smooth layout with reusable Tailwind utility classes
Fast-loading static website

🛠️ Technologies Used
HTML5
Tailwind CSS

🎯 Purpose
This project was created to practice responsive web design and improve front-end development skills using Tailwind CSS. It demonstrates how utility-first CSS can be used to build attractive and maintainable user interfaces efficiently.
